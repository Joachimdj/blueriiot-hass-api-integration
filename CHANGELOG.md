# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added

- Initial release of Blueriot Blue Connect integration
- Support for pool water analyzer device
- Sensor entities for pH, chlorine, temperature, alkalinity, ORP, and conductivity
- Hub configuration with WiFi connectivity
- Configurable polling interval for sensor updates
- Support for API authentication with optional API key
- HACS compatibility
- Multi-language support (English)
- Comprehensive documentation and troubleshooting guide
- Error handling and connection validation
- Data update coordinator for efficient polling

### Features

- Real-time pool water monitoring
- Hub-based device discovery
- Flexible API layer for hub communication
- Proper Home Assistant entity integration
- Device grouping and identification
- State class configuration for history tracking
